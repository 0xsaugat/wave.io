How a neural network learned wave physics.

8 min read ARCHITECTURE & THEORY

The challenge was not to teach a model to mimic water, but to teach it the fundamental rules of propagation. We needed a system that wouldn't just interpolate between known frames, but extrapolate into unseen boundary conditions based on pure physics.

1. The Governing Equation

At the core of our approach is the standard 2D wave equation. Instead of using it to simulate step-by-step, we use it as a penalty. The network must propose a solution, and the equation grades it.

∂²u/∂t² = c² · (∂²u/∂x² + ∂²u/∂y²)

2. PINN Architecture

We employ a Physics-Informed Neural Network (PINN). The network takes spatial coordinates (x, y) and time (t) as inputs, and predicts the wave amplitude (u). By utilizing automatic differentiation, we can compute the exact derivatives of the network's output with respect to its inputs, allowing us to evaluate the wave equation directly on the network's predictions.

3. The Loss Function

The magic lies in the loss function. It's a composite of data loss (where we have ground truth) and physics loss (where the wave equation must hold true).

def physics_loss(model, x, y, t, c):
    u = model(x, y, t)
    
    # Compute gradients using autodiff
    u_t = grad(u, t)
    u_tt = grad(u_t, t)
    
    u_x = grad(u, x)
    u_xx = grad(u_x, x)
    
    u_y = grad(u, y)
    u_yy = grad(u_y, y)
    
    # The PDE residual
    residual = u_tt - (c**2) * (u_xx + u_yy)
    
    return mean_squared_error(residual, 0)

4. Training Dynamics

Training requires careful balancing of the loss terms. Early epochs often see the network ignoring physics in favor of matching the data points. As learning rate schedules engage, the physics loss forces the interpolation to adhere strictly to the wave equation, resulting in physically plausible ripples and interference patterns.

info

Honest Framing

While PINNs offer incredible extrapolation capabilities, they are notoriously difficult to train compared to standard supervised models. Convergence is highly sensitive to the scaling between data loss and physics loss, often requiring dynamic weighting schemes to prevent the gradient of one dominating the other.
